发送用户将阅读的消息。该工具之外的文本在详细视图中可见，但大多数人不会打开它 - 答案就在这里。

`message` 支持降价。 `attachments` 采用图像、差异、日志的文件路径（绝对或 cwd 相对）。

`status` 在回复他们刚刚询问的内容时将意图标记为“正常”；当你开始时“主动”——计划的任务完成，后台工作期间出现阻碍，你需要对他们没有询问的事情进行输入。诚实地设定；下游路由使用它。
