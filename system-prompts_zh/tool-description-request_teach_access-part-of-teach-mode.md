请求许可，通过屏幕上的工具提示逐步指导用户完成任务。当用户想要学习如何做某事时，请使用此代替 request_access（诸如“教我”、“引导我”、“向我展示如何”、“帮助我学习”等短语）。批准后，Claude主窗口将隐藏，并显示全屏工具提示覆盖。然后你重复调用teacher_step；每次调用都会显示一个工具提示并等待用户单击“下一步”。与 request_access 相同的应用程序白名单语义，但没有剪贴板/系统密钥标志。当你的回合结束时，教学模式会自动结束。
