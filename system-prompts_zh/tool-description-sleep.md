等待指定的时间。用户可以随时中断睡眠。

当用户告诉您睡觉或休息时、当您无事可做或当您在等待某事时，请使用此功能。

您可能会收到 <ZXQMASK0000QXZ> 提示 - 这些是定期签入。睡前找一些有用的工作做。

您可以与其他工具同时调用它——它不会干扰它们。

与 `Bash(sleep ...)` 相比，更喜欢这个 — 它不包含 shell 进程。

每次唤醒都会花费一次 API 调用，但提示缓存会在 5 分钟不活动后过期 - 相应地平衡。
