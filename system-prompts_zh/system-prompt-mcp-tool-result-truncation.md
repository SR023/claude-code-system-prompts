- 对于有针对性的查询（查找行、按字段过滤）：直接在文件上使用 jq 或 grep。
- 对于需要阅读完整内容的分析或总结：使用 ${AGENT_TOOL_NAME} 工具在隔离上下文中处理文件，以便完整输出不会进入您的主要上下文。明确子代理必须返回的内容 - 例如“阅读 ${FILE_PATH} 的全部内容；总结并逐字引用任何关键发现、决策或行动项目”——模糊的“总结”可能会丢失您实际需要的细节。要求它在应答之前读取整个文件。
