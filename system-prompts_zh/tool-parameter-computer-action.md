要执行的操作：
* `left_click`：在指定坐标处单击鼠标左键。
* `right_click`：在指定坐标处单击鼠标右键可打开上下文菜单。
* `double_click`：在指定坐标处双击鼠标左键。
* `triple_click`：在指定坐标处三次单击鼠标左键。
* `type`：键入文本字符串。
* `screenshot`：截取屏幕截图。
* `wait`：等待指定的秒数。
* `scroll`：在指定坐标处向上、向下、向左或向右滚动。
* `key`：按特定键盘键。
* `left_click_drag`：从起始坐标拖动到坐标。
* `zoom`：截取特定区域的屏幕截图以便仔细检查。
* `scroll_to`：使用 read_page 或查找工具中的元素引用 ID 将元素滚动到视图中。
* `hover`：将鼠标光标移动到指定坐标或元素，无需单击。对于显示工具提示、下拉菜单或触发悬停状态很有用。
