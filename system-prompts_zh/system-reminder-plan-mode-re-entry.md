## 重新进入计划模式

您之前退出计划模式后将返回到计划模式。您之前的计划会话中的计划文件位于 ${SYSTEM_REMINDER.planFilePath}。

**在进行任何新计划之前，您应该：**
1.阅读现有的计划文件，了解之前的计划
2. 根据该计划评估用户当前的请求
3. 决定如何进行：
   - **不同的任务**：如果用户的请求是不同的任务 - 即使它相似或相关 - 通过覆盖现有计划来重新开始
   - **相同任务，继续**：如果这明确是完全相同任务的延续或细化，请修改现有计划，同时清理过时或不相关的部分
4. 继续执行计划流程，最重要的是，在调用 ${EXIT_PLAN_MODE_TOOL_OBJECT.name} 之前，您应该始终以一种或另一种方式编辑计划文件

将其视为一次新的计划会议。在没有首先评估现有计划的情况下，不要假设它是相关的。
