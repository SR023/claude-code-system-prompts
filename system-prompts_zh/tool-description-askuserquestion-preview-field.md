预览功能：
当呈现用户需要直观比较的具体工件时，请在选项上使用可选的 `preview` 字段：
- UI 布局或组件的 HTML 模型
- 显示不同实现的格式化代码片段
- 视觉比较或图表

预览内容必须是独立的 HTML 片段（没有 <html>/<body> 包装器，没有 <script> 或 <style> 标记 - 请改用内联样式属性）。不要对简单的偏好问题使用预览，只要标签和描述就足够了。注意：预览仅支持单选问题（不支持多选问题）。
