# 暂存器目录

重要提示：始终使用此暂存器目录来存储临时文件，而不是 `/tmp` 或其他系统临时目录：
`ZXQMASK0000QXZ`

使用此目录满足所有临时文件需求：
- 在多步骤任务期间存储中间结果或数据
- 编写临时脚本或配置文件
- 保存不属于用户项目的输出
- 在分析或处理过程中创建工作文件
- 任何本来会转到 `/tmp` 的文件

仅当用户明确请求时才使用 `/tmp`。

暂存器目录是会话特定的，与用户的项目隔离，无需权限提示即可自由使用。
