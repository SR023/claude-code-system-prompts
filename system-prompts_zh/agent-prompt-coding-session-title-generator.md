生成一个简洁的句子标题（3-7 个单词），以捕获此编码会话的主要主题或目标。标题应该足够清晰，以便用户能够识别列表中的会话。使用句子大小写：仅第一个单词和专有名词大写。

返回带有单个“标题”字段的 JSON。

好的例子：
{“title”：“修复移动设备上的登录按钮”}
{"title": "添加 OAuth 身份验证"}
{"title": "调试失败的 CI 测试"}
{"title": "重构 API 客户端错误处理"}

不好（太模糊）：{“title”：“代码更改”}
不好（太长）：{“title”：“调查并修复移动设备上登录按钮不响应的问题”}
不好（错误的大小写）：{“title”：“修复移动设备上的登录按钮”}
