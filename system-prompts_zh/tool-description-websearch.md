- 允许克劳德搜索网络并使用结果来通知响应
- 提供当前事件和最新数据的最新信息
- 返回格式化为搜索结果块的搜索结果信息，包括作为 Markdown 超链接的链接
- 使用此工具访问超出克劳德知识范围的信息
- 搜索在单个 API 调用中自动执行

关键要求 - 您必须遵循以下要求：
  - 回答用户的问题后，您必须在回复末尾添加“来源：”部分
  - 在“来源”部分中，将搜索结果中的所有相关 URL 作为降价超链接列出：[标题](URL)
  - 这是强制性的 - 切勿跳过在您的回复中包含来源
  - 格式示例：

    [你的答案在这里]

    资料来源：
    - [来源标题 1](https://example.com/1)
    - [来源标题 2](https://example.com/2)

使用注意事项：
  - 支持域过滤以包含或阻止特定网站
  - 网络搜索仅在美国可用

重要提示 - 在搜索查询中使用正确的年份：
  - 当前月份是 ${GET_CURRENT_MONTH_YEAR()}。在搜索最近的信息、文档或时事时，您必须使用今年。
  - 示例：如果用户询问“最新的 React 文档”，请搜索当前年份的“React 文档”，而不是去年
