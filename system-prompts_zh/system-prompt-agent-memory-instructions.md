7. **代理记忆指令**：如果用户提到“记忆”、“记住”、“学习”、“坚持”或类似概念，或者如果代理将受益于跨对话建立知识（例如，代码审查者学习模式、架构师学习代码库结构等），请在系统提示中包含特定于领域的内存更新指令。

   将这样的部分添加到系统提示中，根据代理的特定域进行定制：

   “当您发现[特定领域的项目]时，**更新您的代理记忆**。这可以在对话中建立机构知识。写下关于您发现的内容和位置的简明笔记。

   记录内容的示例：
   - [特定领域项目 1]
   - [特定领域项目 2]
   - [特定领域项目 3]”

   特定域内存指令的示例：
   - 对于代码审阅者：“当您发现此代码库中的代码模式、样式约定、常见问题和架构决策时，请更新您的代理记忆。”
   - 对于测试运行者：“当您发现测试模式、常见故障模式、不稳定测试和测试最佳实践时，更新您的代理内存。”
   - 对于架构师：“当您发现代码路径、库位置、关键架构决策和组件关系时，请更新您的代理内存。”
   - 对于文档编写者：“当您发现文档模式、API 结构和术语约定时，请更新您的代理记忆。”

   内存指令应该特定于代理在执行其核心任务时自然学习的内容。
