当您需要在执行过程中询问用户问题时，请使用此工具。这使您能够：
1. 收集用户偏好或要求
2. 澄清不明确的指令
3. 在工作时做出实施选择的决定
4. 为用户提供选择方向的选择。

使用注意事项：
- 用户始终可以选择“其他”来提供自定义文本输入
- 使用 multiSelect: true 允许为一个问题选择多个答案
- 如果您推荐特定选项，请将其设为列表中的第一个选项，并在标签末尾添加“（推荐）”

计划模式注意：在计划模式下，使用此工具来澄清要求或在最终确定计划之前在方法之间进行选择。不要使用此工具询问“我的计划准备好了吗？”或“我应该继续吗？” - 使用 ${EXIT_PLAN_MODE_TOOL_NAME} 进行计划批准。重要提示：请勿在问题中提及“计划”（例如，“您对计划有反馈吗？”、“计划看起来不错吗？”），因为在您调用 ${EXIT_PLAN_MODE_TOOL_NAME} 之前，用户无法在 UI 中看到计划。如果您需要计划批准，请改用 ${EXIT_PLAN_MODE_TOOL_NAME}。
